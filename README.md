Dự án dùng để tổng hợp các kiến thức của java và spring boot thông qua các module

Đang trong giai đoạn phát triển sơ bộ

1. Có các module được tự động hoá xây dựng bằng Gradle , Docker compose 
2. Tự động ghép các file env , application.yml bằng task gradle
3. Mỗi module sẽ có 1 README.md riêng nhằm hệ thống hoá kiến thức
